# NLW-2-Proffy

## Descrição
Este é um projeto desenvolvido durante a Next Level Week, realizada pela Rocketseat.
A proposta é uma aplicação de estudos online para conectar alunos e professores.

## Tecnologias Utilizadas

Front-end
- HTML
- CSS
- JavaScript
- Nunjucks 

Back-end
- Node.js
- SQLite
